# BBC iPlayer - UK TV streaming client

**Package**: `@apps/iplayer`
**Version**: 0.1.0
**Repository**: `pkg-apps-iplayer`

## Overview

BBC iPlayer - UK TV streaming client

## Installation

### Prerequisites

No external dependencies required.

### Using pkg-cli

```bash
# Install from GitHub
pkg-cli install @apps/iplayer

# Or install from local source
cd ~/.pkgs
stow iplayer
```

## Usage

### Application

BBC iPlayer - UK TV streaming client

Run the application to see usage instructions.

## Configuration

Configuration files are typically stored in:
- `~/.config/iplayer/` - User configuration
- `~/.local/share/iplayer/` - Application data

## Uninstallation

```bash
# Using pkg-cli
pkg-cli uninstall @apps/iplayer

# Or manual unstow
cd ~/.pkgs
stow -D iplayer
```
